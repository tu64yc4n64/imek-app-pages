# İMEK – Gizlilik & Destek sayfaları

Bu klasörü **ayrı bir repo** olarak oluşturup GitHub Pages ile yayınlayın.

## Adımlar

1. GitHub’da yeni bir repo oluşturun (ör. `imek-app-pages` veya `imek-privacy`).
2. Sadece bu klasördeki dosyaları o repo’ya push edin (`index.html`, `support.html`).
3. Repo → Settings → Pages → Source: **Deploy from a branch** → Branch: `main` → `/ (root)` → Save.
4. Yayınlandıktan sonra URL’ler:
   - **Gizlilik politikası:** `https://<kullanici-adiniz>.github.io/<repo-adi>/`
   - **Destek:** `https://<kullanici-adiniz>.github.io/<repo-adi>/support.html`

## Mağaza linkleri

- **Privacy Policy URL:** `https://<kullanici-adiniz>.github.io/<repo-adi>/`
- **Support URL:** `https://<kullanici-adiniz>.github.io/<repo-adi>/support.html`

`support.html` içindeki `[destek e-posta adresinizi buraya yazın]` kısmını kendi e-posta adresinizle değiştirin.
